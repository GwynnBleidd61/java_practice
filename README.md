📁 Структура папок

HelloWorld — самые базовые примеры: синтаксис языка, ввод/вывод, первые программы.

OOP — задачи на объектно-ориентированное программирование: классы, наследование, интерфейсы, инкапсуляция.

algorithms — тренировка алгоритмов и структур данных: массивы, списки, стеки, деревья, графы, сортировки.

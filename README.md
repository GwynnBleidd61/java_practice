## Структура репозитория

- **HelloWorld**  
  Самые базовые примеры:
  - синтаксис Java;
  - переменные, типы данных, операции;
  - ввод/вывод;
  - первые консольные программы.

- **OOP**  
  Задачи на объектно-ориентированное программирование:
  - классы и объекты;
  - инкапсуляция, наследование, полиморфизм;
  - абстрактные классы и интерфейсы;
  - базовые принципы проектирования.

- **Collections**  
  Практика коллекций из Java Collections Framework:
  - `List`, `Set`, `Map`, `Queue`;
  - работа с итераторами;
  - сравнение разных реализаций (ArrayList/LinkedList, HashSet/TreeSet и т.д.);
  - простые задачи на обработку коллекций.

- **algorithms**  
  Тренировка алгоритмов и структур данных:
  - массивы, списки, стеки, очереди;
  - деревья и графы (по мере развития);
  - сортировки и поиск;
  - простые задачки в стиле собеседований.
